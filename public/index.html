Welcome to the official admin page, this page contains access points that allow you to make changes to database content.

You can...
Create a resource
create Resource database for both new and old students (that have not been previously registered).
list a resource
in order to read a particular result to verify correctness, or to list all the student resources in the database.
Update a resource
to update existing student resource or to efffect changes or to correct errors in the resource database.
Delete a resource
this deals with the intricacies of deleting resource(S) from the database, resources deleted are permanently lost.

Enter the admission number of students whose resource you want to read.

Enter the admission number of students whose resource you want to delete.

The only compulsory field is the admission number, which filters the resource, and apply the changes to the file with that specific admission number.
